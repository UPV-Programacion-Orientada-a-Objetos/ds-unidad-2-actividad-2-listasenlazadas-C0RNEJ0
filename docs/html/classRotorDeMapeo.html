Detailed Description

Lista circular doblemente enlazada que actúa como disco de cifrado.

Implementa un mecanismo similar a la Rueda de César donde la rotación del rotor cambia el mapeo de caracteres. El alfabeto A-Z está dispuesto en forma circular.
